# MedCore - Hospital Management System

## VS Code me Run karne ke Steps (Easy Guide)

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 ya usse upar) install karo
- [VS Code](https://code.visualstudio.com/) install karo

### Step 1: Project Clone / Download karo
```bash
# Agar GitHub se clone kar rahe ho:
git clone <your-repo-url>
cd <project-folder>
```

### Step 2: VS Code me folder open karo
```bash
code .
```

### Step 3: Terminal kholo VS Code me
- **Shortcut:** `Ctrl + `` ` (backtick key)
- Ya menu se: **Terminal → New Terminal**

### Step 4: Dependencies install karo
```bash
npm install
```

### Step 5: App run karo
```bash
npm run dev
```

### Step 6: Browser me dekho
Terminal me link aayega: **http://localhost:8080**
Ctrl+Click karke browser me open karo.
